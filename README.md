# PyTorch-for-Deep-Learning-Coursera-Course

This repository contains hands-on labs, assignments, and projects completed while studying PyTorch through DeepLearning.AI's PyTorch specialization.

## Topics Covered

### Course 1: PyTorch Fundamentals

* Tensors and tensor operations
* Neural networks
* Activation functions
* Data management and DataLoaders
* MNIST classification
* Convolutional Neural Networks (CNNs)
* Model debugging

### Course 2: PyTorch Techniques & Ecosystem

* Hyperparameter tuning
* Learning rate schedulers
* Optuna optimization
* Model efficiency and profiling
* TorchVision
* Transfer learning
* NLP tokenization
* Word embeddings
* Text classification
* Fine-tuning pretrained models

## Repository Structure

* `PytorchFundamentals-course1/` – Fundamentals labs and assignments
* `PytorchTechniques&Ecosystems-course2/` – Advanced techniques, computer vision, and NLP labs

## Skills Demonstrated

PyTorch, Deep Learning, Neural Networks, CNNs, Computer Vision, Transfer Learning, NLP, Hyperparameter Optimization, Model Profiling, TorchVision, and Hugging Face workflows.
